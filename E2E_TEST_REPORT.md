## 4. 测试发现

### 无 Critical 问题
E2E 测试未发现阻塞性问题，demo.html 的所有页面和交互功能均正常工作。

### 观察到的设计细节
1. 工作流运行中步骤有 CSS 脉冲动画（`@keyframes pulse`）
2. 智能体卡片悬停时有紫色发光阴影效果
3. 看板卡片悬停时有上浮 + 阴影加深效果
4. 通知列表第一条为安全警告（JWT 默认密钥）
5. 设置页面安全配置区域标注"需要关注"
